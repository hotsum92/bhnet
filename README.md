# bhnet

Network client and Server program.

[2.5 Netcatの置き換え (BOOK: サイバーセキュリティプログラミング Pythonで学ぶハッカーの思考)]()

## example

Open a command prompt.

```
python ./bhnetserver.py -p 22
```

Open another command prompt.

```
python ./bhnet.py -t localhost -p 22
```

Start BHP prompt.

```
<BHP:#> ls -al
```
